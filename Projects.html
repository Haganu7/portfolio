Project:Space War(Scratch)
Type of project:Partner Project
Course:Computer Science Principles
Duration:August 2018-September 2018
Objective:Make a game in Scratch with the given time limit

The first partner project that we worked on in my Computer Sciences Principles course. The objective of the project was to create a game in Scratch. It could be any type of game and we could design it as we see fit. Then we would have to add information about the project into our online journal. This being my first project I had learned a lot. The most important lesson I learned is that creating any type of code is time consuming. You really have to plan out your time well or you might not finish on time. My role in the group was mainly to document information in our online journal while my partners role was to code. The biggest challenge I faced was the lack of communication between my partner and I due to only one person being able to edit at a time it was rather hard to edit it off school grounds. Overall I think both my partner and I's teamwork was really good. While in class we could both work on two seperate things at a time and ended up finishing things faster than if we were working alone.
